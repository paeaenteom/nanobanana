# 🍌 NanoBanana — AI 이미지 생성기

Google Gemini API의 Nano Banana 모델을 사용한 AI 이미지 생성 웹앱.

## 기능
- 프롬프트 입력 → AI 이미지 생성
- 참고 이미지 업로드 (최대 14장) + 이름 태그
- 모든 Nano Banana 모델 선택 가능
- 갤러리 보기 + PNG 다운로드
- 비율 선택 (1:1, 16:9, 9:16 등)

## 사용법
1. [Google AI Studio](https://aistudio.google.com/apikey)에서 무료 API 키 발급
2. 사이트에서 키 입력
3. 프롬프트 작성 → 🍌 생성!

## 배포
Vercel에 바로 배포 가능:
```bash
vercel --prod
```

## 비용
Gemini API 무료 티어 사용 — **무료!**
